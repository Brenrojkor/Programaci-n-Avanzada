# Pokémon

A functional Pokémon game/project developed with the tools seen in the Advanced Programming course.

## Features

- **Pokedex.**
- **Users.**
- **Nursing.**
- **Fighting Team.**
- **Coaches.**
- **Messages.**
- **Login.**
- **Sign-Up.**

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** C#
- **Database:** SQL Express
- **Server:** MVC
- **Version Control:** Git
